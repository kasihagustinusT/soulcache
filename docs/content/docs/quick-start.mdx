## Quick Start

This guide will get you up and running with SoulCache in under 5 minutes.

<Steps>
  <Step title="Install SoulCache">
    ```bash
    npm install @soulcache/core
    ```
  </Step>

  <Step title="Create a Query Client">
    ```typescript
    import { QueryClient } from '@soulcache/core';

    const client = new QueryClient({
      defaultOptions: {
        staleTime: 5 * 60 * 1000, // 5 minutes
        gcTime: 30 * 60 * 1000, // 30 minutes
      },
    });
    ```
  </Step>

  <Step title="Fetch Data">
    ```typescript
    // Fetch with automatic caching
    const data = await client.fetchQuery({
      queryKey: ['users'],
      queryFn: () => fetch('/api/users').then(r => r.json()),
    });
    ```
  </Step>

  <Step title="Subscribe to Changes">
    ```typescript
    // Subscribe to query updates
    const unsubscribe = client.subscribe(['users'], (snapshot) => {
      console.log('Data updated:', snapshot.data);
      console.log('Status:', snapshot.status);
    });
    ```
  </Step>

  <Step title="Update Cache">
    ```typescript
    // Optimistically update cache
    client.setQueryData(['users'], (old) => [...old, newUser]);

    // Invalidate and refetch
    await client.invalidateQueries(['users']);
    ```
  </Step>

  <Step title="Cleanup">
    ```typescript
    // Cleanup when done
    client.destroy();
    ```
  </Step>
</Steps>

## Complete Example

Here's a complete example putting it all together:

```typescript
import { QueryClient } from '@soulcache/core';

async function main() {
  // Create client
  const client = new QueryClient();

  // Define a query
  const users = await client.fetchQuery({
    queryKey: ['users'],
    queryFn: async () => {
      const response = await fetch('https://jsonplaceholder.typicode.com/users');
      return response.json();
    },
  });

  console.log('Users:', users);

  // Subscribe to changes
  const unsubscribe = client.subscribe(['users'], (snapshot) => {
    console.log('Update received:', snapshot.data);
  });

  // Update cache
  client.setQueryData(['users'], (old) => [
    ...old,
    { id: 4, name: 'New User' },
  ]);

  // Cleanup
  unsubscribe();
  client.destroy();
}

main();
```
