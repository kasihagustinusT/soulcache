## QueryEngine

The QueryEngine is an internal orchestrator that manages query lifecycle, caching, and observer coordination. It is accessed indirectly through `QueryClient`.

### Access

```typescript
import { QueryClient } from '@soulcache/core';

const client = new QueryClient();
// QueryEngine is used internally — use QueryClient methods directly
```
