We welcome contributions to SoulCache!

### Getting Started

<Steps>
  <Step title="Fork and Clone">
    Fork the repository on GitHub, then clone your fork:
    ```bash
    git clone https://github.com/your-username/soulcache.git
    cd soulcache
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    pnpm install
    ```
  </Step>
  <Step title="Create a Branch">
    ```bash
    git checkout -b feature/my-feature
    ```
  </Step>
  <Step title="Make Changes and Test">
    Make your changes, then run the test suite:
    ```bash
    pnpm test
    ```
  </Step>
  <Step title="Commit and Push">
    ```bash
    git commit -m "feat: add my feature"
    git push origin feature/my-feature
    ```
  </Step>
  <Step title="Create a Pull Request">
    Open a pull request on GitHub with a clear description of your changes.
  </Step>
</Steps>

### Code Style

<Note type="info" title="Coding Standards">
- Use TypeScript with strict mode
- Follow existing patterns in the codebase
- Write tests for new features
- Update documentation for API changes
</Note>

### Pull Request Guidelines

<Note type="tip" title="PR Checklist">
- Write clear, descriptive commit messages
- Include tests for new features
- Update documentation as needed
- Follow existing code style and conventions
</Note>
